Shared Express + WS Server ‚ö°

In real apps, WebSocket and HTTP live on the same port. Learn how to mount a WebSocket server on an existing Express app ‚Äî sharing port 3000 for both REST and real-time.

1
The Problem ‚Äî two separate servers
Real apps share one port for HTTP and WebSocket
‚ùå Separate servers (tutorial style)
// HTTP on :3000, WS on :8080 ‚Äî TWO ports
app.listen(3000);
new WebSocketServer({ port: 8080 });

Two ports = two firewall rules, two SSL certs. Don't do this.

‚úÖ Shared server (production style)
// Both on :3000 ‚Äî ONE port
const server = http.createServer(app);
const wss = new WebSocketServer({ server });
server.listen(3000);

One port, one SSL cert, professional.

2
Full Setup ‚Äî complete server.js üíª
Express REST API + WebSocket on the same port 3000
</> server.js ‚Äî complete shared server
const express = require('express');
const http = require('http');
const { WebSocketServer } = require('ws');

const app = express();
app.use(express.json());

// REST routes (HTTP)
app.get('/api/status', (req, res) => {
  res.json({ clients: wss.clients.size });
});
app.post('/api/broadcast', (req, res) => {
  wss.clients.forEach(c => c.readyState === 1 && c.send(req.body.msg));
  res.json({ sent: wss.clients.size });
});

// Shared HTTP server
const server = http.createServer(app);
const wss = new WebSocketServer({ server });

wss.on('connection', (ws, req) => {
  // req.socket, req.headers, cookies ‚Äî all available!
  ws.send(JSON.stringify({ type: 'welcome' }));
});

server.listen(3000, () => console.log('Port 3000'));
Key: Pass the HTTP server object into WebSocketServer, not a port number. Call server.listen() once ‚Äî it handles both HTTP and WS.
3
Path-based WS routing üîÄ
Multiple WebSocket endpoints on the same port
</> Route /ws and /admin separately
const wss = new WebSocketServer({ noServer: true });
const admin = new WebSocketServer({ noServer: true });

server.on('upgrade', (req, socket, head) => {
  if (req.url === '/ws')
    wss.handleUpgrade(req, socket, head, ws => wss.emit('connection', ws, req));
  else if (req.url === '/admin')
    admin.handleUpgrade(req, socket, head, ws => admin.emit('connection', ws, req));
  else
    socket.destroy(); // reject unknown paths
});
